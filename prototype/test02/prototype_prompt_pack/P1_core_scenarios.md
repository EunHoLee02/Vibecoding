# P1 단계 프로토타입 핵심 시나리오 정의

## 1. 프로토타입 핵심 사용자 시나리오

### 시나리오 S1. 직접 입력 후 분석
- 시작 조건: 로그인 없이 첫 화면에 진입, 사용자는 복용 중인 영양제 정보를 직접 입력하려고 함
- 사용자 행동:
  1. 직접 입력 시작 버튼 또는 입력 영역 선택
  2. 영양제 1개 이상 정보 입력
  3. 필요 시 영양제 추가 입력
  4. 분석 실행 버튼 클릭
  5. 결과 화면에서 분석 결과 확인
- 시스템 반응:
  - 입력값 수집
  - 복수 영양제 조합을 rule-based 방식으로 분석
  - 중복 성분, 과다 가능성, 복용 시간 가이드, 한 줄 요약 반환

### 시나리오 S2. 샘플 이미지/샘플 데이터 선택 후 분석
- 시작 조건: 로그인 없이 첫 화면 진입, 직접 입력 대신 샘플 업로드/샘플 선택으로 기능 체험
- 사용자 행동:
  1. 샘플 이미지 업로드 또는 샘플 데이터 선택 영역 열기
  2. 샘플 항목 1개 이상 선택 또는 업로드
  3. 시스템이 인식된 영양제 목록 또는 mock 추출 결과 표시
  4. 분석 실행 버튼 클릭
  5. 결과 화면에서 분석 결과 확인

### 시나리오 S3. 입력값 수정 후 재분석
- 시작 조건: 한 번 분석 결과를 본 상태
- 사용자 행동:
  1. 결과 화면 또는 입력 화면으로 돌아감
  2. 영양제 1개를 추가/삭제하거나 입력값 수정
  3. 분석 실행 버튼 다시 클릭
  4. 변경된 결과 확인

## 2. 시연용 대표 사용자 플로우
- 대표 플로우 A: 직접 입력 → 분석 실행 → 결과 확인
- 대표 플로우 B: 샘플 업로드/샘플 선택 → 분석 실행 → 결과 확인
- 대표 플로우 C: 분석 완료 → 입력 수정 → 재분석 → 변경 결과 확인

## 3. 필수 사용자 행동 목록
1. 로그인 없이 시작
2. **직접 입력** 또는 **샘플 이미지/샘플 데이터 선택** 중 하나 선택
3. 영양제 항목 1개 이상 입력/선택
4. 필요 시 영양제 추가
5. **분석 실행** 버튼 클릭
6. 결과 화면에서 아래 항목 확인
   - 입력된 영양제 목록
   - 중복 성분 목록
   - 과다 가능성 표시
   - 복용 시간 가이드
   - 요약 한 줄
7. 필요 시 입력 수정 후 다시 분석

## 4. 필수 시스템 반응 목록
1. 익명 단일 사용자 흐름으로 동작
2. 직접 입력값 또는 샘플 기반 입력값을 분석 대상 목록으로 구성
3. 업로드 입력의 경우 실제 OCR 대신 **stub/mock 처리**
4. 분석 실행 시 **rule-based 우선** 방식으로 결과 계산
5. 결과 화면에서 아래 항목 반환
   - 입력 영양제 목록
   - 중복 성분
   - 과다 가능성
   - 복용 시간 가이드
   - 전체 요약 한 줄
6. 결과를 **일반 건강 가이드 수준**으로 표현
7. 의료 진단, 처방, 치료 보장처럼 해석될 수 있는 문구 사용 금지
8. 수정된 입력값으로 재분석 가능
9. 사용자별 누적 저장 없이도 흐름 완결

## 5. P0 / P1 / P2 시나리오 우선순위
- P0: `S1_manual_input_analysis`, `S2_sample_upload_analysis`
- P1: `S3_edit_and_reanalyze`, 업로드 인식 결과 보정, 결과 설명 강화
- P2: 결과 저장, 개인화, 관리자/운영

## 6. 제외해야 할 시나리오
- 로그인/회원가입
- 비밀번호 재설정
- 관리자 검토/승인/수정
- 결제/구독 시작
- 문의/오류 신고
- 사용자별 기록 누적/마이페이지
- 장기 복용 이력 저장/비교
- 의료 진단/처방/치료 권고
- 실제 OCR 정확도 검증 중심 시나리오
- 배포/운영/모니터링 중심 시나리오

## 7. 다음 단계에서 바로 사용할 수 있는 시나리오 고정 요약
- 대표 사용자 흐름은 **입력 → 분석 실행 → 결과 확인**
- 입력 방식은 **직접 입력**과 **샘플 이미지/샘플 데이터 선택** 두 가지 필수 지원
- 분석은 **rule-based 우선**, 업로드 인식은 **stub/mock 허용**
- 결과 화면은 `입력 목록 / 중복 성분 / 과다 가능성 / 복용 시간 가이드 / 전체 요약 한 줄` 포함
- 사용자는 필요 시 입력 수정 후 **재분석** 가능
