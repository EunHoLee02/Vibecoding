[출력 고정 규칙]
- 섹션 제목과 순서를 그대로 유지할 것
- 비어 있는 섹션은 생략하지 말고 `없음`으로 적을 것
- 동일 성격의 목록은 별도 우선순위가 없으면 오름차순으로 정렬할 것
- authoritative_updates YAML은 반드시 ```yaml fenced block```으로 감쌀 것

[authoritative_updates YAML 템플릿]
authoritative_updates:
  frozen_values:
    key: value
  supersedes:
    - old_source -> new_source
  deprecated:
    - deprecated_key_or_rule
  carry_forward_only:
    - key1
    - key2
  do_not_carry_forward:
    - key3
    - key4

stage_status:
  maturity: draft|provisional|minimum_complete|fully_verified
  unresolved_conflicts:
    - none
  missing_p0_items:
    - none
